JVI API

The Java Versant Interface (JVI) implements the full object model. Methods are provided to define embedded, association and inheritance relationships, either at compile time or at runtime.

JVI also implements complete Versant database functionality as both functions and utilities. JVI functionality is available through the standard Java.

For more information, please refer to the Versant Java Interface Manual.

If you are new to the Versant Object Database technology we strongly recommend to start with the Tutorial.

Examples

Query:

IDE

Advanced Examples

DataBaseUtilities

Shows how to use the JVI interface for making your own database tools.

Event

Two example applications using the Event Notification package. Although these programs can be run individually, they work together using the event notification mechanism.

Fund

A collection of examples demonstrating how to use the fundamental JVI interface for different tasks.

JCA

An example for how to use the Versant JCA Connector with a WebLogic application server.

MemoryManagement

Demonstrates how to query information about the databases memory and how to manage it yourself.

ObjectMigrationCopying

Shows how objects can be moved from one database to another.

Sharing

A demonstration of how to share database objects between different programming languages (Java/C++).

UserAuthentification

Two short example applications that demonstrate how to create database users and how to authenticate to a database.

XA

Two examples that demonstrate the use of Versants JTA/XA package.


Note: To run the demos on 64bit platforms, please add following line to the target element:

<jvmarg value="-d64"/>
