Persistence Classification Wizard

The Persistence Classification Wizard enables the user to classify the different Java classes in the application into persistence categories that are meaningful to Versant Database. Upon invocation, this tool shows a basic dialog box, in which classes can be marked as either not-persistent, or as persitent-capable. These basic categories should be sufficient for a large number of applications. For applications that need advanced categories of persistence, clicking the ‘Advanced’ button of the Persistence Classification Wizard will present the advanced dialog box, where all possible categories are shown. These categories are described in detail in The JVI User’s Guide. The Persistence Classification Wizard saves its categorization information in the config.jvi file. The config.jvi file is the input to the Enhancer tool, which uses it to enhance the classes.

Enhancer Wizard

The Enhancer Wizard is a tool that enhances the class files of an application to provide transparent persistence capabilities. It uses the config.jvi file to determine the type of enhancement that needs to be performed on each class file. The Enhancer has a variety of options that can be set, which are described in The JVI User’s Guide.

 

Versant Object Database Tools

  • Monitoring Console

  • Object Inspector

  • Administration Console

The Monitoring Console is a database On-Line Tuning Tool. The Administration Console is used for creation, deletion and other operations on the database. The Object Inspector is used for browsing through classes and their objects, query, and other related operations. These tools come up in their own windows, and have help text available with them.
